# Live2D 动作和表情控制演示

## 🎯 新功能概览

我已经成功为你的 Live2D Recorder 项目添加了完整的动作和表情控制功能！

## ✨ 主要特性

### 1. 🎬 动作控制系统
- **分类显示**: 动作按类型自动分组（angry、smile、thinking等）
- **一键播放**: 点击按钮即可播放对应动作
- **状态反馈**: 当前播放的动作会高亮显示
- **随机动作**: 点击模型可播放随机动作
- **参数控制**: 支持设置动画索引和优先级
  - 动画索引: 控制动作的播放顺序
  - 优先级: 控制动作的播放优先级（0-10）

### 2. 😊 表情控制系统
- **表情切换**: 支持所有可用的表情状态
- **实时预览**: 显示当前使用的表情
- **分类管理**: 表情按类型分组显示

### 3. 🎛️ 智能控制面板
- **可折叠设计**: 支持隐藏/显示，不遮挡模型
- **美观界面**: 半透明黑色主题，现代化设计
- **响应式布局**: 自动适应不同屏幕尺寸
- **状态显示**: 实时显示当前动作和表情状态
- **参数设置**: 动画索引和优先级控制

## 🎮 使用方法

### 基本操作
1. **启动应用**: 运行 `npm run dev` 或 `npm run tauri:dev`
2. **显示控制面板**: 右上角会显示控制面板
3. **播放动作**: 点击动作分类下的按钮
4. **切换表情**: 点击表情分类下的按钮
5. **隐藏面板**: 点击右上角的 ✕ 按钮

### 高级控制
1. **动画索引**: 在"动画参数"区域设置动画索引（通常为0）
2. **优先级设置**: 设置动作播放的优先级（0-10，默认3）
3. **参数应用**: 所有动作播放都会使用当前设置的参数

### 交互方式
- **点击模型**: 播放随机动作
- **点击按钮**: 播放特定动作或切换表情
- **隐藏/显示**: 控制面板可折叠，不影响观看
- **参数调整**: 实时调整动画参数，立即生效
