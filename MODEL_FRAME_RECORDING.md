# 模型边框录制功能使用指南

## 概述

模型边框录制功能是一个性能优化特性，它只录制Live2D模型范围内的内容，而不是整个画布。这可以显著减少文件大小、提升录制性能，并减少后期处理的工作量。

## 功能特性

### 🎯 智能区域录制
- **自动边框计算**: 根据模型尺寸和位置自动计算最优录制区域
- **性能优化**: 只录制模型范围内的内容，减少不必要的像素处理
- **文件大小优化**: 录制文件更小，传输和存储更高效

### 🖼️ 录制区域可视化
- **边框显示**: 可选的绿色虚线边框，实时显示录制范围
- **尺寸信息**: 显示录制区域的宽度、高度和位置坐标
- **动态调整**: 支持录制过程中动态调整录制区域

### ⚡ 性能提升
- **CPU使用率降低**: 减少需要处理的像素数量
- **内存占用减少**: 录制缓冲区更小
- **导出速度提升**: 文件处理更快

## 使用方法

### 1. 启用模型边框录制

1. 在控制面板中找到"🎥 录制设置"部分
2. 勾选"启用模型边框录制优化"复选框
3. 系统会自动计算并显示模型的录制边框

### 2. 显示录制边框

1. 勾选"显示录制区域边框"复选框
2. 页面上会显示绿色虚线边框，标识录制区域
3. 边框会显示录制区域的尺寸和位置信息

### 3. 开始录制

1. 在时间线中添加动作或表情片段
2. 点击录制按钮开始录制
3. 系统会自动使用模型边框录制器
4. 录制过程中会显示绿色边框（如果启用）

### 4. 调整录制区域

如果需要手动调整录制区域：

```typescript
// 在代码中动态调整录制区域
if (recRef.current && 'updateModelBounds' in recRef.current) {
  recRef.current.updateModelBounds({
    x: 100,
    y: 100,
    width: 400,
    height: 300
  });
}
```

## 技术实现

### 核心函数

```typescript
export function createModelFrameRecorder(
    canvas: HTMLCanvasElement,
    modelBounds: { x: number; y: number; width: number; height: number },
    fps = 60,
    kbps = 16000,
    options?: {
        autoStop?: boolean;
        onProgress?: (time: number) => void;
        transparent?: boolean;
        showFrame?: boolean; // 是否显示录制边框
    }
)
```

### 边框计算逻辑

```typescript
// 自动计算模型边框
const modelWidth = model.width * model.scale.x;
const modelHeight = model.height * model.scale.y;
const modelX = model.position.x - modelWidth / 2;
const modelY = model.position.y - modelHeight / 2;

setModelBounds({
  x: Math.max(0, modelX),
  y: Math.max(0, modelY),
  width: Math.min(modelWidth, app.screen.width),
  height: Math.min(modelHeight, app.screen.height)
});
```

### 录制流程

1. **初始化**: 创建裁剪canvas和边框显示
2. **流创建**: 使用`canvas.captureStream()`获取视频流
3. **录制**: 通过MediaRecorder进行VP9/VP8编码录制
4. **清理**: 录制完成后自动清理资源

## 注意事项

### 浏览器兼容性
- 需要支持VP9/VP8编码的现代浏览器
- Chrome/Edge推荐使用
- Firefox可能需要VP8格式

### 性能考虑
- 模型边框录制适合模型相对固定的场景
- 如果模型频繁移动，建议使用全屏录制
- 录制过程中避免大幅调整模型位置

### 最佳实践
1. **录制前**: 确认模型位置和尺寸合适
2. **录制中**: 避免大幅移动模型
3. **录制后**: 检查录制区域是否完整

## 故障排除

### 录制区域不正确
- 检查模型是否正确加载
- 确认模型位置和缩放设置
- 尝试刷新页面重新计算边框

### 性能提升不明显
- 确认模型尺寸相对于画布较小
- 检查是否启用了边框显示
- 验证录制器类型是否正确

### 边框不显示
- 确认勾选了"显示录制区域边框"
- 检查控制台是否有错误信息
- 验证模型边框计算是否成功
